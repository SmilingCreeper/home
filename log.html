MinecraftOS更新日志

b1.0（不开放下载）

MinecraftOS Beta 1.0是MinecraftOS首个内测版本，更新于2023年3月20日17:47。此版本加入一些基础元素，例如开机时的转圈动画、任务栏等，这两个元素一直被保留到现在。

b1.1（不开放下载）

MinecraftOS Beta 1.1是MinecraftOS的第二个内测版本，更新于2023年3月20日17:51。此版本加入开始菜单、背景图片等。开始菜单一直被保留至今。

MinecraftOS Release 1.0（不开放下载）

MinecraftOS 正式版1.0是MinecraftOS首个正式版，更新于2023年3月21日09:32，加入了Minecraft教程（没什么用，已停止维护），但是当时教程的帧率很低。

MinecraftOS Release 2.0（不开放下载）

MinecraftOS 正式版2.0更新于2023年5月25日19:59，加入了小病毒（可以让MinecraftOS蓝屏），此版本的开始菜单无法取消。

MinecraftOS Release 3.0

MinecraftOS 正式版3.0更新于2023年7月15日19:04，是首个开放下载的版本，更换了桌面背景图片，加入了音量调节功能。

MinecraftOS Release 3.1

MinecraftOS 正式版3.0更新于2023年7月20日20:26，加入桌球小游戏。

MinecraftOS Release 3.2

MinecraftOS 正式版3.0更新于2023年7月22日20:54，加入点击按钮的音效和小彩蛋（点击右下角SmilingCreeper按钮即可触发）
